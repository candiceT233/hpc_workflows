OVERVIEW
--------
This directory contains downloaded test input data for 8 nf-core Nextflow
pipelines.  Each pipeline has two scales:

  small/   -- Minimal CI test data (from conf/test.config)
  medium/  -- Scaled-up data (from conf/test_full.config where HTTPS-accessible;
              otherwise 3x concatenation of small FASTQ data)

Data was downloaded from https://github.com/nf-core/test-datasets (various
branches) and from NCBI where needed for reference genomes.

IMPORTANT NOTES:
- Medium-scale data for most pipelines was generated by 3x concatenation of
  small FASTQ files, because the test_full.config profiles reference AWS S3
  buckets (s3://ngi-igenomes, s3://nf-core-awsmegatests) that are not directly
  accessible via HTTPS without AWS credentials.
- The samplesheet CSVs/TSVs downloaded into medium/ still contain the original
  S3 URLs from test_full.config for reference; they cannot be used directly.
- To use this data with the pipelines, you will need to create local
  samplesheets pointing to the downloaded file paths.
- No containers are available on this system; tools must be installed natively
  or via conda/modules.
